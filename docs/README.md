# HOLP Docs Index

Start here before reading individual docs. Current authority is split by level:
`README.md` for repo status, `docs/holp-blueprint.md` + `docs/roadmap.md` for
route, and `docs/pr-specs/README.md` for PR-level specs.

| Path | Use |
| --- | --- |
| `docs/holp-blueprint.md` | Current authoritative M7-M12 route and completion contracts. |
| `docs/roadmap.md` | Milestone roadmap and current implementation facts from M0 through M12. |
| `docs/positioning.md` | Product/protocol positioning, non-goals, and source-of-design boundaries. |
| `docs/holp-multiround-base-spec.md` | Multiround / step-loop base spec for the M7 foundation. |
| `docs/holp-adjustments-spec.md` | WorkPlanner and learned-router preparation adjustments. |
| `docs/holp-learned-router-framework.md` | Directional learned-router framework and sequencing rationale. |
| `docs/runtime-session-matrix.md` | Runtime/session vocabulary and direct-session capability notes. |
| `docs/runtime-surface-validation-matrix.md` | Issue #52 validation matrix and #45 gate record. |
| `docs/2026-06-22-backlog-priority.md` | Historical backlog snapshot; not the current route authority. |
| `docs/pr-specs/` | PR-level specs. Open specs stay at the root; completed specs live in `docs/pr-specs/completed/`. |
| `docs/audits/` | Documentation audit and reorganization reports. |
| `docs/notes/` | Working notes and process ledgers that are not normative specs. |
